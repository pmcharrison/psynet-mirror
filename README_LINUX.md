# Installation on Ubuntu 20.04

### Following files are required to be modified in Dallinger:

##### build-requirements.txt:
* Add 'cython' package

##### constraints-deps.txt:
* Use 'numpy==1.17.5'
* Use 'packaging==20.3'
* Use 'psycopg2==2.8.2'
* Use 'pandas==1.1.3'
* Use 'pyyaml==5.3.1'
* Use 'virtualenv==20.1.0'
